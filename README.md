# Donna-Data-Visualizer

有一个在生物实验室研究生的Donna，每周都要开组会，老板要检查她的实验记录本，而她已经欠下了好几个月的试验记录没有做，还要自己作图，真的很麻烦 T.T

那么，我们帮她实现一个方便快速的作图工具吧！这样Donna就可以很方便的写好实验记录本，通过老板的检查啦！

# 使用说明

1. 在选择文件处选择一个表格文件xls、asc或csv
2. 点击plot初步生成图片
3. 根据需要输入x轴、y轴、曲线图标题
4. 选择是否需要标出峰值
5. 如果有两条曲线并且只需要显示其中的一条，可以进行选择
6. 在图片中的曲线上点击相应的位置，可以自主标注峰值（如果标错，可以再次点击plot刷新）
7. 当对曲线图有改动（步骤3-5）后，需要重新点击plot生成新的图片
8. 所有工作完成后，确定曲线图无误则点击save保存图片



就这样，可爱的Donna顺利完成了实验记录本，开开心心的啦！
